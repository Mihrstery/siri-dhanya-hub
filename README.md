# Siri Dhanya Hub

A modern Android application built with **Kotlin** and **Jetpack Compose**, featuring a clean architecture and contemporary UI design.

## 📱 Overview

Siri Dhanya Hub is an Android application that demonstrates best practices in modern Android development. It leverages the latest Android technologies including Jetpack Compose for declarative UI, Kotlin coroutines, and Material Design 3.

## 🚀 Features

- **Modern UI with Jetpack Compose** - Declarative and intuitive user interface
- **Material Design 3** - Latest Material Design system with Material Icons
- **Image Loading** - Integrated Coil for efficient image loading and caching
- **Navigation** - Jetpack Navigation Compose for seamless screen transitions
- **Android Best Practices** - Built with modern Android development standards

| Technology | Version | Purpose |

## 📋 Requirements
